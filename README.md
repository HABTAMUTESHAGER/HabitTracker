# Habit Tracker Pro - Final Project

## Project Overview
Habit Tracker Pro is a comprehensive web application designed to help users build and maintain positive habits through tracking, visualization, and analytics.

## Features
- **Dashboard**: Overview of daily progress, streaks, and completion rates
- **Habits Management**: Create, edit, and track habits with custom frequencies
- **Statistics**: Detailed insights with charts and progress tracking
- **Analytics**: Advanced metrics and habit performance analysis
- **Settings**: Customizable theme, reminders, and data management

## Technologies Used
- **HTML5**: Semantic structure with 6 separate pages
- **CSS3**: Responsive design with Flexbox/Grid, transitions, and variables
- **JavaScript**: DOM manipulation, Chart.js integration, local storage
- **Chart.js**: Data visualization for statistics and analytics
- **Font Awesome**: Icon library for UI elements
- **Google Fonts**: Inter font for typography

## Key Features Implementation
1. **Single Page Application (SPA) Navigation**: Smooth transitions between sections
2. **Responsive Design**: Works on desktop, tablet, and mobile devices
3. **Data Persistence**: Local storage for habits and settings
4. **Interactive Charts**: Weekly overview, category distribution, progress tracking
5. **Theme Customization**: Light/dark mode with accent colors
6. **Form Validation**: Habit creation with time validation
7. **Data Import/Export**: JSON and CSV support
8. **Reminder System**: Browser notifications for habits

## Project Structure
The application follows a modular structure with separate HTML files for each main section, connected through a main navigation page.

## How to Run
1. Download all project files
2. Open `index.html` in any modern web browser
3. No server or installation required

## Browser Compatibility
